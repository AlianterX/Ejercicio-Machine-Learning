# Ejercicio-Machine-Learning
Concurso de datos Pump it Up: Data Mining the Water Table

Utilizando datos de Taarifa y del Ministerio de Agua de Tanzania, ¿puedes predecir qué bombas funcionan, cuáles necesitan reparaciones y cuáles no funcionan en absoluto? Esta es una competencia práctica de nivel intermedio. Predice una de estas tres categorías basándote en diversas variables sobre el tipo de bomba en funcionamiento, su fecha de instalación y su gestión. Un conocimiento preciso de qué puntos de agua fallarán puede mejorar las operaciones de mantenimiento y garantizar que las comunidades de Tanzania tengan acceso a agua potable.

Estas son las variables del concurso: https://www.drivendata.org/competitions/7/pump-it-up-data-mining-the-water-table/page/25/

Puntuacion mas alta obtenida: 0.8107
